# OCR Document MVP

A lightweight OCR utility for extracting text from image files.

## Current Status

M1 Bootstrap complete.

## Features

- Validate image path
- Support PNG, JPG, JPEG, WEBP
- CLI command: `ocr-doc`
- Pytest coverage
